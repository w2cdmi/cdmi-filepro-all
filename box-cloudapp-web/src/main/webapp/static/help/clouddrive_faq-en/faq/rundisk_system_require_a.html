What Are the Requirements of StorBox on the System?

StorBox Synchronization Client

You can run StorBox synchronization client in Windows. The requirements on the system are as follows:
  • Windows 7 32-bit operating system
  • Memory cannot be lower than 512 MB.
  • Space on the computer cannot be less than that in the synchronization folder.

StorBox Website

You can use Windows Internet Explorer 8 or a later version to access StorBox.

StorBox Mobile Applications

You can use a mobile phone running Android OS 4.0 or a later version (such as Huawei Mate and Huawei P6) to access StorBox.
