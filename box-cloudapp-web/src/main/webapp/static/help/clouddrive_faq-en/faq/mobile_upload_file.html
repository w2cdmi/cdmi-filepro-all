How Can I Upload a File from My Mobile Phone to StorBox?

Click the add button on the upper right of the screen and select the file format you want to add. You can create a folder and upload documents, photos, and all data in your mobile phone. After the data is uploaded successfully, StorBox synchronizes all data to your computer or other devices.
