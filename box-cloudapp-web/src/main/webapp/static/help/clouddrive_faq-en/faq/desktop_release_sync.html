How Can I Cancel the Synchronization Between the Synchronization Client and the Cloud?

If the synchronization between the synchronization client and the cloud is canceled, data synchronization between the synchronization folder and the cloud is mutually stopped.

Canceling synchronization using the synchronization client

Right-click the tray icon of the synchronization client and select Pause Sync.

Canceling synchronization using the website

  1. Log in to StorBox website.
  2. Hover the cursor over the file or folder and click on the right.
