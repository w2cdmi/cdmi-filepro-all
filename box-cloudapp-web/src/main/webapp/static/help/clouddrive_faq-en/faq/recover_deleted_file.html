How Can I Recover a Deleted File?

You can recover a deleted file using the StorBox website.

  1. Log in to StorBox website.
  2. Click Recycle Bin.
  3. Right-click the file or folder and choose Restore.
    NOTE:

    Alternatively, you can click the blank area to the right of the file or folder and choose Restore above the file list.
