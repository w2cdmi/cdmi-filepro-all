How Can I Delete a File in StorBox?

You can delete files in StorBox using the synchronization client in the same way as deleting local files. Alternatively, you can log in to the StorBox website to delete the files, and the files are deleted from other devices on which StorBox is installed.

Deleting a file using the synchronization client

Remove the file you want to delete from the synchronization folder or delete it to the Windows recycle bin.

Files that you have deleted using the synchronization client are saved in Recycle Bin on the website.

Deleting a file using the website

  1. Log in to StorBox website.
  2. Select the file or folder you want to delete and drag it to the recycle bin.

    If you want to delete the file permanently, click Recycle Bin to go to the recycle bin page and delete the file.

    NOTE:

    Alternatively, you can right-click the blank area to the right of the file or folder and choose Delete or choose More > Delete above the file list.
