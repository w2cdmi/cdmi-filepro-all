Why Should I Download Files or Folders in My Mobile Phone?

After a file or folder is downloaded, is displayed on the icon of the file or folder. Items downloaded are saved in the mobile phone. Accessing the data does not consume downloading traffic.

Files or folders downloaded in the mobile phone consume mobile phone space. If the mobile phone space becomes insufficient, delete some of the files or folders to release more space.
