Can I Run Multiple StorBox Users on One Computer?

Only one StorBox can be installed on a computer and only one user can be associated with a StorBox.

If another user wants to use the StorBox synchronization client running in the same computer, the association of the running user to the computer must be deleted first.
  1. Right-click the StorBox tray icon and select Options.
  2. On the Settings tab, click Unlink this computer.

    After the association is deleted, the previous user's data is reserved on the computer.
