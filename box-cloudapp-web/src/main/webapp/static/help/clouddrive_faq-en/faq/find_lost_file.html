How Can I Get a Lost File Back?

A file may get lost due to many factors. You can get a lost file back by performing the following steps:
  • Check whether the file is shared with other users.

    If the file disappears, the file sharer has removed you from the recipients or the sharer has deleted the shared file or has canceled the share. Contact the sharer to confirm the reason.

  • Locate the file by searching for it.

    Log in to StorBox website and enter the name of the lost file in the search field. You may find that the file has been moved to another location.

  • Check the recycle bin for the file.

    If a file is deleted mistakenly on a terminal device, the same file on other terminals is also deleted because of the synchronization function. You can log in to StorBox website and check the recycle bin for the file. If you find the file, you can recover it to its previous location.

If the previous methods fail, report the problem here along with the information about your location (for example, the US or Shenzhen), name of the lost file, creation time, and file type.
