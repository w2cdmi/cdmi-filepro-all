What File Formats Can Be Displayed on My Mobile Phone?

Files in the StorBox for Android are encrypted for storage to ensure data security. If you want to open a file, you must install the WPS Office client specified by Huawei.

The WPS Office client supports the following file formats:
  • Text fiels (.txt)
  • Word (.doc/.docx/.dot/.dotx)
  • Excel (.xls/.xlsx/.xlt/.xltx)
  • PowerPoint (.ppt/.pptx/.pps/.ppsx)
  • PDF （.pdf）

The StorBox for Android integrates a plug-in for browsing pictures in the bmp, gif, jpg, and jpeg formats.
