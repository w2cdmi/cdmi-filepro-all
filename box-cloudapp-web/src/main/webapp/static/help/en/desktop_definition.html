What Is StorBox Synchronization Client?

<<Back to Help Center

StorBox synchronization client is the software that enables file synchronization between local disks and cloud in Windows. You can access StorBox using the synchronization folder on the computer.

In addition to using the StorBox website, you are advised to download and use StorBox synchronization client.

Why? Because the synchronization client is more convenient to use. You can use StorBox in the same way as using other directories in local disks, except that data in synchronization folder is automatically synchronized to other devices related to the user.

The synchronization client automatically monitors folder status to ensure that all files are synchronized. If the computer is connected to the Internet, all your changes to files are synchronized to other devices on which StorBox has been installed.
