How Can I Cancel a File or Folder Share?

<<Back to Help Center

If you are the sharer, you can cancel the share anytime.

After you cancel the share, the shared file or folder is removed from other recipients' Received shares page. Other recipients cannot recover the file or folder.

Perform the following steps to cancel a file or folder share:

  1. Log in to StorBox website.
  2. Hover the cursor over the file or folder and click on the right.
    Note

    Alternatively, you can right-click the blank area to the right of the file or folder and choose Share or click Share above the file list to go to the Share window.

  3. Click Cancel File Sharing or Cancel Folder Sharing.
