What Is StorBox Cloud Manager Client?

<<Back to Help Center

The StorBox cloud manager client is software running on Windows to provide file upload, download, sharing, collaboration, and backup services.

In addition to using the StorBox website, you are advised to download and use StorBox cloud manager client.

Why? The reason is that the StorBox cloud manager client can help you quickly upload big files, implement resumable data transfer, and back up disks, constantly protecting the security of local files.
