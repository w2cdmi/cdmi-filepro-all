How Can I Synchronize Files Among Computers?

<<Back to Help Center

StorBox supports file synchronization among multiple terminals (computers or mobile terminals). You need to install StorBox synchronization client in all terminals and log in to all terminals as the same user.

After logging in to StorBox synchronization client, you can upload files or change files in the synchronization folder, and the synchronization client synchronizes all new changes to StorBox. You can view the latest files in any computer or mobile terminal on which StorBox is installed.

You can share files or folders with other users.

Installing StorBox synchronization client

  1. Download StorBox synchronization client to all the computers whose data needs to be synchronized.
  2. Install the synchronization client as prompted.

    The synchronization client is installed in C:\Program Files\StorBox\xxx by default. xxx indicates the version of the synchronization client.

    After running the synchronization client for the first time, you need to set the directory in which the synchronization folder is saved. You can set a local folder that is not empty as the synchronization folder, and data in this folder will be automatically uploaded to StorBox. By default,StorBox automatically creates a synchronization folder named StorBox in the disk partition with the most available space. You can change the name and location of the synchronization folder.

After StorBox synchronization client is installed, it automatically monitors the synchronization folder without manual intervention to ensure that you can obtain the latest files anytime and anywhere.
