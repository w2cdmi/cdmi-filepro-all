How Can I Set the Number of Concurrent Transfer Tasks on the StorBox Cloud Manager Client?

<<Back to Help Center

By default, the StorBox cloud manager client supports three concurrent transfer tasks. You can adjust the value based on site conditions.

Large number of concurrent transfer tasks indicate slow response speed of the local PC. Small number of concurrent transfer tasks indicate long task execution time.

You can right-click the icon of the client and choose Settings > Transfer to change the number of concurrent transfer tasks.
