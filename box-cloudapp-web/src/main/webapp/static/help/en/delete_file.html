How Can I Delete a File in StorBox?

<<Back to Help Center

You can log in to the StorBox website or StorBox cloud manager client to delete the files.

Deleting a file using the website

  1. Log in to StorBox website.
  2. Right-click the file or folder and choose Delete in the right-click menu to delete the file or folder.
    Note

    Alternatively, you can select the file or folder and choose More > Delete above the file list to delete the file or folder.

    If you want to delete the file or folder permanently, click Recycle Bin icon to go to the recycle bin page and delete it.

Deleting a file using the StorBox cloud manager client

You can right-click the file or folder and choose Delete in the right-click menu to delete the file or folder.

Note

Alternatively, you can select the file or folder and choose More > Delete above the file list to delete the file or folder.

Files that you have deleted using the StorBox cloud manager client are saved in Recycle Bin on the website.

If you want to delete files permanently, click Recycle Bin icon to go to the recycle bin page and delete them.
