How Can I Know that the Synchronization Client Is Synchronizing Data?

<<Back to Help Center

The synchronization client tray icon is in the system taskbar. If the two rotating arrows disappear from the tray icon, the synchronization client has finished the synchronization.

Pause synchronization
Synchronizing
Synchronization completed

Besides, there are two states for the tray icon:

Offline

This icon appears when the user is not logged in to the synchronization client, synchronization client is disabled, or the user cannot connect to the synchronization client.

Authentication failure

This icon appears if login authentication fails.
