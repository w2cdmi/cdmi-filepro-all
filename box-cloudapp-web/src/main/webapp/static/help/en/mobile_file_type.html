What File Formats Can Be Displayed on My Mobile Phone?

<<Back to Help Center

StorBox for Android

If WPS Office has been installed on an Android phone, StorBox for Android can open files in the following formats:

  • Text files (.txt)
  • Word (.doc/.docx)
  • Excel (.xls/.xlsx/.xlt/.xltx)
  • PowerPoint (.ppt/.pptx)
  • PDF (.pdf)

The StorBox for Android integrates a plug-in for browsing pictures in the bmp, gif, jpg, png, and jpeg formats.

StorBox for iOS

StorBox for iOS can open files in the following formats:

  • Text files (.txt)
  • Word (.doc/.docx/.dotx)
  • Excel (.xls/.xlsx/.xlt/.xltx)
  • PowerPoint (.ppt/.pptx)
  • PDF (.pdf)

The StorBox for iOS integrates a plug-in for browsing pictures in the bmp, gif, jpg, png, and jpeg formats.
