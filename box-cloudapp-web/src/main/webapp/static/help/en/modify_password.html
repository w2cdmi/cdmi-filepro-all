How to Modify the password?

<<Back to Help Center

It is recommended to change the password periodically for your data security.You can log in to the StorBox website to change your password.

  1. Log in to the StorBox website.
  2. Click the user name on the top-right and click Settings.
  3. Click Change password in the User information page table.
  4. Enter the old password, new password, and new password again to confirm. Then click Edit.
