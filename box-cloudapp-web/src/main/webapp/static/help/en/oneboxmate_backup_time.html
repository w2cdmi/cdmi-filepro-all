What Is the Backup Window of the StorBox Cloud Manager Client?

<<Back to Help Center

After the backup function is enabled, the StorBox cloud manager client automatically scans the local backup directory every ten minutes. If new data is written to the local backup directory, the StorBox cloud manager client backs up new data to the cloud, continuously safeguarding file security.
