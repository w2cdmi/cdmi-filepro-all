Can I Run My StorBox Synchronization Client on Multiple Computers?

<<Back to Help Center

Yes. After downloading and installing StorBox synchronization client on multiple computers, you can log in to the synchronization client in a computer to upload, download, and synchronize files. Changes to the files in the folder are synchronized to all computers on which StorBox synchronization client is installed.
