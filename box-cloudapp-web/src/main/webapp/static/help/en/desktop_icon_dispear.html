What Can I Do If There Is No Icon on the File or Folder?

<<Back to Help Center

If there is no icon on the file or folder, the reason is probably that the data fails to be written into the registration table during the synchronization client installation. Perform the following steps to solve the problem:

  1. Run the cmd.exe command as an administrator to enter the command-line interface (CLI).
  2. Run the cd /d C:\Program Files\StorBox command to enter the installation directory of the synchronization client. Suppose that the synchronization client is installed in disk C and the installation directory is C:\ProgramFiles\StorBox.
  3. Run the regsvr32 ShellExtent.dll command to manually register shell expansion.
  4. Run the following commands to manually restart the Resource Manager.
    1. taskkill /f /im explorer.exe
    2. explorer.exe
