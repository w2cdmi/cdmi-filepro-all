How Can I Create Team Space?

<<Back to Help Center

  1. Log in to StorBox website.
  2. In the left navigation tree, click Team Space.
  3. Click Create.

  4. Enter a name for the team space.
  5. (Optional) Enter description for the team space and select Message notification for file upload.

    After this option is selected, notification messages are sent to team members if data in the team space is changed.

    Team members can click in the upper right area of the StorBox interface to view notification messages.

  6. Click OK.
