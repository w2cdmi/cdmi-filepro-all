How Can I Share a File or Folder with Other Users?

<<Back to Help Center

StorBox enables you to share a file or folder with other users. Two methods are available:

  • Link

    Create a link for a file or folder, and then send the link to your friends or colleagues to share photos, documents, or videos with them. Your friends or colleagues can open the link in a browser to download the file or folder.

    How Can I Share a File or Folder Using a Link?

  • Share

    Create a share for a file or folder to share documents and photos with other StorBox users. Changes in the shared data are automatically synchronized to other users.

    How Can I Share a File or Folder?

Using a Link

  • Website
    1. Log in to StorBox website.
    2. Hover the cursor over the file or folder and click on the right.

      Alternatively, you can use the following method to open the Share Link window.

    3. Set permission on the right of Access permissions.
      • When a file is shared, the recipients have download and preview permission.
      • When a folder is shared, the recipients have upload, download and preview permission.
    4. Set a value to Extraction code and Validity period, and click Save.

      Extraction code is the password used by a user to view the link.
      By default, there is no extraction code in the system, and the link is valid permanently.

    5. Click , enter the recipients, then click Send.
      Alternatively, you can copy the link and the extraction code to the notepad. Then you can send the link and the extraction code in the notepad to other recipients using other methods.
  • Windows Cloud Manager Client
    1. Right-click the tray icon of StorBox cloud manager client and choose Show StorBox.
    2. Hover the cursor over the file or folder and click on the right in the My Space page.
    3. Click Create Share Link, and set the access permission, extraction code and validity period, then click Finish.

      Extraction code is the password used by a user to view the link.
      By default, there is no extraction code in the system, and the link is valid permanently.

    4. Copy the link and the extraction code, and send them to other recipients.

Using Share

  • Website
    1. Log in to StorBox website.
    2. Hover the cursor over the file or folder and click on the right.

      Alternatively, you can use the following method to open the Share window.

    3. Enter a recipient's information, and select a recipient's access permission, then click Invite.

      If the invitation is successful, the folder turns and becomes a shared folder.

  • Windows Cloud Manager Client
    1. Right-click the tray icon of StorBox cloud manager client and choose Show StorBox.
    2. Hover the cursor over the file or folder and click on the right in the My Space page.
    3. Enter a recipient's information, and select a recipient's access permission, then click Invite.
