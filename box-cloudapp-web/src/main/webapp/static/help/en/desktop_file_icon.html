What Do the Icons Indicate?

<<Back to Help Center

The synchronization client monitors the status of the synchronization folder in real time to ensure that data in the folder is synchronized with that on the StorBox website. The icon on the tray icon and the icon on the file or folder indicate what is happening in the file or folder.

  • The icon on the tray icon indicates the overall status of synchronization folder.
  • The icon on the file or folder indicates the status of the file or folder.

The following icons are displayed:

Synchronization completed
A blue circle with a tick in the middle is a common icon.

It indicates that the latest synchronization of the file or folder is complete.

Synchronizing
A blue circle with two rotating arrows is the synchronization icon.

  • If this icon appears on the tray icon, the synchronization client is synchronizing files (uploading or downloading files).
  • If this icon appears on the file or folder, it indicates that this file or folder is being synchronized.

Abnormal synchronization
The underlined grey circle indicates a synchronization failure.

If this icon appears on the file or folder, the refresh of the local file/folder cannot be synchronized to the cloud because the cloud cancelled file/folder synchronization or the file/folder was deleted from the cloud.

If no icon appears on the file or folder, refer to What Can I Do if There Is No Icon on the File or Folder?
