Can I Rename or Move a Shared File or Folder?

<<Back to Help Center

You can rename or move a shared file or folder in the same way as renaming or moving other directories on the computer. Alternatively, you can rename or move a file or folder using StorBox website or cloud management client.

Renaming or moving a shared file or folder does not change its shared status. The location of the shared file or folder is not changed and its name turns to the new name.
