How Can I Cancel a File or Folder Link?

<<Back to Help Center

If you are the sharer of a file or folder link, you can cancel the link anytime.

After you cancel the link, Invalid link is displayed when other recipients try to open it.

Perform the following steps to cancel a file or folder link:：

  1. Log in to StorBox website.
  2. Hover the cursor over the file or folder and click on the right.
    Note

    Alternatively, you can right-click the blank area to the right of the file or folder and choose Share Link or click Share Link above the file list to go to the Share Link window.

  3. Click Unshare.
