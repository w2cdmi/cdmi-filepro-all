What Are the Requirements of StorBox on the System?

<<Back to Help Center

StorBox Cloud Manager Client

You can run StorBox cloud manager client in Windows. The requirements on the system are as follows:

  • Windows 7 32-bit/64-bit operating system
  • Windows 8 32-bit/64-bit operating system

StorBox Website

You can use Windows Internet Explorer 9.0/10.0/11.0 to access StorBox.

StorBox Mobile Applications

You can use a mobile phone running iOS 7.0 or a later version, or a mobile phone running Android OS 4.0 or a later version (such as HUAWEI Mate 7 and SAMSUNG GALAXY S5) to access StorBox.
