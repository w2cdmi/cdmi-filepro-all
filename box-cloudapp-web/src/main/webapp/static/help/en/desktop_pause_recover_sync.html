How Can I Pause and Resume Synchronization?

<<Back to Help Center

The synchronization client can pause and resume synchronization. After the synchronization is paused, data synchronization between the synchronization folder and the cloud is mutually stopped.

Right-click the tray icon and choose Pause Sync or Resume Sync.
