How Is My Privacy Ensured?

<<Back to Help Center

StorBox places great emphasis on your privacy. If you do not share files or file links, others cannot view the files you have saved in StorBox. Even StorBox developers cannot view the data in the files you have saved. When you turn to developers for technical support, they can only access the metadata (such as file names and locations).

We have adopted several measures to ensure your privacy.

  • Network isolation

    The network planes on which the administrator and you reside are isolated from each other. The administrator cannot access the network on which you save files.

  • Slice-based data storage

    After object data is sliced, data slices are stored on disks of different storage nodes randomly based on a distributed hash algorithm. Malicious users cannot obtain user data from a single storage node or disk. When the lifecycle of a service ends or a user stops using a service, the slice-based data storage technology supports secure data destruction, preventing data leak.

  • User data isolation

    Each user owns independent logical space and follows unique storage policies. Data of one user is isolated from that of another user, ensuring data security and privacy.

  • Encrypted storage of sensitive user data

    Sensitive and private user data, such as authentication information and email addresses, is encrypted using the Advanced Encryption Standard (AES) algorithm before being stored.
