How to Set the Data of StorBox Automatically Download to the Computer Which Installed the StorBox Synchronization Client?

<<Back to Help Center

The system shutdown automatical downloading function to save computer capacity when installing the StorBox synchronization client. If your computer has enough capacity, you can set the data of StorBox automatically download to the Computer which installed the StorBox synchronization client. You can use the automatical downloading function as follows:

  1. Right-click the StorBox synchronization client tray icon and select Options.
  2. Click the Settings tab.
  3. Select Automatical download in the Download policy area.
