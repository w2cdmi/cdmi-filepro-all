Why Some Directories Cannot Be Selected As a Backup Directory?

<<Back to Help Center

Backup rules specify that some directories such as system disk directories and software installation directories, for example, Program Files, cannot be backed up to the cloud. Backup rules are configured by enterprise administrators in the user access management system.
