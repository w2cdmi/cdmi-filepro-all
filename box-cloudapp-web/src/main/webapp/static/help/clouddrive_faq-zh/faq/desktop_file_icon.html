图标含义是什么

同步客户端会实时监控同步文件夹的状态，保证文件夹中的内容与StorBox网站中的内容保持同步。同步客户端托盘图标、文件或文件夹上叠加的图标会告诉您发生了什么变化。

  • 同步客户端托盘图标上叠加的图标，显示了同步文件夹的总体状态。
  • 文件或文件夹上叠加的图标，显示了文件或文件夹的状态。
您能看到的所有状态图标如下：

同步完成

带勾号的蓝色圆圈是比较常见的图标。

当文件或文件夹上出现该图标时，表示文件或文件夹已经完成最新更改的同步。

正在同步

带双向旋转箭头的蓝色圆圈是同步图标。

  • 当同步客户端托盘图标上出现该图标时，表示同步客户端正在同步（正在上传文件、下载文件）。
  • 当文件或文件夹上出现该图标时，表示文件或文件夹正在进行同步。

同步异常

带横线的灰色圆圈是无法同步图标。

文件正在等待上传或正在上传到云端时，云端删除了文件或文件所在的文件夹导致本地内容无法与云端同步。

当文件或文件夹上没有显示任何状态图标时，请查看为什么状态图标没有显示解决。
