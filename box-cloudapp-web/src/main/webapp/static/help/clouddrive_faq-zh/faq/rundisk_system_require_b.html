StorBox对系统有哪些要求

StorBox同步客户端

您可以使用Windows操作系统运行StorBox同步客户端。要求如下：
  • Windows 7 32位操作系统
  • 内存不低于512MB
  • 电脑上的空间不低于StorBox同步文件夹的空间

StorBox网站

您可以使用Windows Internet Explorer 8及以上版本的浏览器访问StorBox。

StorBox移动应用

您可以使用Android OS 4.0及以上版本的手机（如Huawei Mate、Huawei P6）访问StorBox。
