如何与他人分享文件或文件夹

StorBox可以让您轻松分享文件或文件夹。这里有两种分享方式：
  • 链接

    您可以为文件或文件夹创建链接，将链接发送到朋友或同事的邮箱，与他们一起分享照片、文档、视频等。您的朋友或同事在浏览器中使用链接地址就可以下载文件和文件夹。

    如何通过链接分享文件或文件夹

  • 共享

    您可以将文件或文件夹共享给其他StorBox用户，与他们共享团队文档和照片。共享内容发生变化后，系统会自动同步给他们。

    如何共享文件或文件夹

链接方式

  • 网站
    1. 登录StorBox网站。
    2. 将光标悬浮在文件或文件夹上，单击右侧的。

      您也可以通过以下方式打开“分享链接”窗口。

    3. 单击“访问限制”右侧的“修改”，设置提取码和有效期，单击“保存”。

      提取码是用户查看链接内容的密码。

      系统默认没有提取码，链接永久有效。

    4. 填写收件人信息，单击“发送”。

      如果您希望通过其他方式分享链接，请复制链接地址和提取码到记事本。通过其他方式将链接和提取码发送给被分享人。

  • Windows
    1. 右键单击托盘图标，选择“打开同步文件夹”。
    2. 右键单击文件或文件夹，选择“StorBox > 分享链接”。
    3. 单击“访问限制”右侧的“修改”，设置提取码和有效期，单击“保存”。

      提取码是用户查看链接内容的密码。

      系统默认没有提取码，链接永久有效。

    4. 复制链接和提取码，发送给被分享人。

共享方式

  • 网站
    1. 登录StorBox网站。
    2. 将光标悬浮在文件或文件夹上，单击右侧的。

      您也可以通过以下方式打开“共享”窗口。

    3. 填写共享人信息，单击“邀请”。

      邀请成功后，文件夹变为，表示该文件夹为共享文件夹。

  • Windows
    1. 右键单击托盘图标，选择“打开同步文件夹”。
    2. 右键单击文件或文件夹，选择“StorBox > 共享”。
    3. 填写共享人信息，单击“邀请”。
