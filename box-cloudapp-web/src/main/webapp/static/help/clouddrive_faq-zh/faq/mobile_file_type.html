我可以在手机上查看哪些类型的文件

为了保证数据安全，StorBox Android应用中的文件采用加密存储。如果您想打开文件，需要安装公司指定的WPS Office客户端。

WPS Office客户端支持以下文件格式：
  • 文本文件（.txt）
  • Word（.doc/.docx/.dot/.dotx）
  • Excel（.xls/.xlsx/.xlt/.xltx）
  • PowerPoint（.ppt/.pptx/.pps/.ppsx）
  • PDF（.pdf）

此外，StorBox Android应用集成了图片浏览插件，支持浏览bmp、gif、jpg、jpeg格式的图片。
