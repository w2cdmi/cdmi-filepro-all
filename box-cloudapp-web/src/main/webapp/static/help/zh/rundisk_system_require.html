StorBox云管家客户端

您可以使用Windows操作系统运行StorBox云管家客户端。要求如下：

  • Windows 7 32/64位操作系统
  • Windows 8 32/64位操作系统

StorBox网站

您可以使用Windows Internet Explorer 9.0/10.0/11.0版本的浏览器访问StorBox。

StorBox移动应用

您可以使用iPhone iOS 7.0及以上版本的手机或者Android OS 4.0及以上版本的手机（如华为Mate7、三星Galaxy S5）访问StorBox。
