如何取消文件或文件夹分享

<<返回帮助中心

如果您是分享文件或文件夹的拥有者，您可以随时取消分享。

当您取消分享后，分享文件或文件夹将从被分享人的"收到的分享"网页中被彻底删除。被分享人无法恢复文件。

取消文件或文件夹分享的操作如下：

  1. 登录StorBox网站。
  2. 将光标悬浮在文件或文件夹上，单击右侧的。
    说明

    您也可以右键单击文件或文件夹的空白处，在右键菜单中，选择“共享”或者单击文件列表上方的“共享”，打开“共享”窗口。

  3. 单击“取消文件共享”或“取消文件夹共享”。
