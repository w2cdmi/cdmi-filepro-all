如果我的硬盘空间无法容纳整个StorBox中的内容，是否可以进行选择性同步

<<返回帮助中心

StorBox默认是将所有文件同步到您的电脑中。

如果StorBox中存储的内容已经超出电脑硬盘的可用空间，您可以在网站上取消部分文件的同步，同时在本地删除这些文件。

  1. 登录StorBox网站。
  2. 将光标悬浮在文件或文件夹上，单击右侧的。

    说明

    您也可以右键单击文件或文件夹的空白处，在右键菜单中，选择"取消同步"或者选择文件列表上方的“更多>取消同步”。
