如何管理团队空间中的成员

<<返回帮助中心
  1. 登录StorBox网站。
  2. 单击左侧导航中的“团队空间”。
  3. 右键单击需要管理的团队空间，选择“成员管理”。

    您也可以单击需要管理的团队空间，在上方的菜单栏选择“成员管理”。

  4. 在“成员管理”窗口中，可以进行如下操作。
    • 查看已有成员及权限。
    • 添加新成员：输入用户的登录名、姓名、邮箱或者群组名称，按回车并设置权限。
    • 删除已有成员：单击成员所在行的。
    • 修改已有成员的权限：单击成员角色右侧的下拉箭头，设置新的权限。
