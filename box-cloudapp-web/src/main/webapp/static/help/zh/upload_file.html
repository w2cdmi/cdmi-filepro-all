如何添加或上传文件

<<返回帮助中心

网站

  1. 登录StorBox网站。
  2. 单击“上传”。

  3. 在打开的窗口中，选中需要上传的文件，单击“打开”。

    在网站右下角显示文件上传进度。上传成功后，界面右下角的上传列表框自动消失，文件已经添加到StorBox中。

Windows

当在电脑上安装StorBox云管家客户端后，您可以通过如下方法将文件上传至云端。

  1. 登录云管家客户端。
  2. 在“个人空间”界面单击“上传”。
  3. 选择需要上传的文件或文件夹，单击“上传”即可。
