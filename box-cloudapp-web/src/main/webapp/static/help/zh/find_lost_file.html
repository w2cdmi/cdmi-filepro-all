我的文件丢失了，怎样才能找回

<<返回帮助中心

导致文件丢失的原因多种多样，请根据下文内容逐一排查，尽快找回文件：

  • 确认丢失的文件是否是别人分享的文件

    如果文件已经消失，可能是文件拥有者将您从分享成员中移除了，也可能是删除了分享文件或者取消了分享。请联系分享拥有者确认原因。

  • 通过搜索找到文件

    请登录StorBox网站，在搜索框中输入丢失文件的名称。通过搜索，发现文件可能被移动到了其他位置。

  • 通过回收站找到文件

    如果文件被误删，您可以登录StorBox网站，在回收站中查找文件。发现文件后，可以将文件恢复到原有位置。

如果通过以上方法，仍然无法找回文件，请将您所在的区域（如深圳、美国）、丢失文件的名称、创建时间反馈给技术支持工程师，工程师将协助您找回文件。
