团队空间有几种角色

<<返回帮助中心

企业管理员可以根据实际需要，在UAM中设置用户访问团队空间的权限种类（拥有者和管理者为系统默认权限，企业管理员无法设置）。

用户最多有七种访问团队空间的权限，说明如下。

  • 拥有者可以解散团队、变更团队拥有者、管理团队，可以浏览、预览、上传、下载、编辑和分享链接。
  • 管理者可以管理团队成员，可以浏览、预览、上传、下载、编辑和分享链接。
  • 编辑者可以浏览、预览、上传、下载、编辑和分享链接。
  • 可上传可查看者可以浏览、预览、上传和下载文档。
  • 查看者可以浏览、预览和下载文档。
  • 预览者仅能预览和浏览文档。
  • 上传者可以浏览和上传文档。
