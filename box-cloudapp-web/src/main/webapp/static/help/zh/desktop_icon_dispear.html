当文件或文件夹上没有显示任何状态图标时，如何解决

<<返回帮助中心

当文件或文件夹上没有显示任何状态图标时，可能是安装同步客户端时写注册表失败造成的。请参考下文的方法解决。

  1. 以管理员身份运行cmd.exe，进入命令行界面。
  2. 运行cd /d C:\Program Files\StorBox进入同步客户端安装目录。此处以同步客户端安装在C盘，安装目录是C:\ProgramFiles\StorBox为例。
  3. 运行regsvr32 ShellExtent.dll手动注册shell扩展。
  4. 运行以下命令手动重启资源管理器。
    1. taskkill /f /im explorer.exe
    2. explorer.exe
