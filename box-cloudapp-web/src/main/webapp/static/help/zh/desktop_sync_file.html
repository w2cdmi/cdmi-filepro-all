如何在电脑之间同步文件

<<返回帮助中心

在多终端（电脑、移动终端）之间同步文件是StorBox的特性。您只需要安装StorBox同步盘客户端，并使用同一用户登录所有终端。

登录StorBox同步盘客户端后，您在同步文件夹中添加文件或修改文件，同步盘客户端会自动将最新内容同步到StorBox，您可以在任何一台安装了StorBox的电脑、移动终端上查看最新更新的文件。

如果您想与其他StorBox用户共享文件，您可以共享文件或文件夹给其他用户。

安装StorBox同步盘客户端

  1. 在每一台需要同步信息的电脑上，下载StorBox同步盘客户端。
  2. 根据界面提示安装同步盘客户端。

    默认安装在"C:\Program Files\Huawei\Onebox\xxx"中。xxx表示同步盘客户端版本号。

    第一次运行同步盘客户端时，您需要设置同步文件夹。您可以设置本地非空文件夹为同步文件夹，非空文件夹中的内容自动上传到StorBox。否则软件默认在剩余空间最多的硬盘分区中创建一个名为“Onebox“的同步文件夹，您可以更改同步文件夹的名称和位置。

安装StorBox同步盘客户端后，您无需操作任何按钮，设置任何项目，同步盘客户端会自动监控同步文件夹，确保您随时随地都可以获得最新更新的文件。
