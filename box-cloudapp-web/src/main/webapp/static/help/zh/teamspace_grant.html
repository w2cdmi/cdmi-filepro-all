如何将团队空间中的某个文件夹设置为授权文件夹，他人可以在该文件夹中上传或编辑文档

<<返回帮助中心
  1. 登录StorBox网站。
  2. 单击左侧导航中的“团队空间”。
  3. 双击团队空间图标或者单击团队空间的名称进入团队空间。
  4. 右键单击需要设置的文件夹，选择“文件夹授权”。

  5. 在“文件夹授权”窗口中，单击“单击添加更多成员”。
  6. 选择需要添加的成员，并设置其权限。
  7. 单击“确定”。

    根据需要，重复步骤5-步骤6继续添加成员。

  8. 单击“关闭”。
