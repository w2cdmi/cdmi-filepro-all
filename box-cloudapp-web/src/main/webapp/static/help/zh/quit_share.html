如何退出分享

<<返回帮助中心

当退出分享后，您将无法重新加入，但分享文件或文件夹仍然在其他分享用户的StorBox中。如果您是分享文件或文件夹的拥有者，您可以取消文件或文件夹的分享、删除分享文件或文件夹中的成员。

退出分享的操作如下：

  1. 登录StorBox网站。
  2. 选择“收到的共享”。
  3. 右键单击文件或文件夹，选择“退出共享”。
    说明

    您也可以单击文件或文件夹的空白处，在文件列表上，选择“更多>退出共享”。
