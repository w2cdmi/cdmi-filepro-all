如果我的硬盘空间无法容纳整个Huawei CloudDrive中的内容，是否可以进行选择性同步

<<返回

Huawei CloudDrive可以让您轻松分享文件或文件夹。这里有两种分享方式：

  • 链接

    您可以为文件或文件夹创建链接，将链接发送到朋友或同事的邮箱，与他们一起分享照片、文档、视频等。您的朋友或同事在浏览器中使用链接地址就可以下载文件和文件夹。

    如何通过链接分享文件或文件夹

  • 共享

    您可以为文件或文件夹创建链接，将链接发送到朋友或同事的邮箱，与他们一起分享照片、文档、视频等。您的朋友或同事在浏览器中使用链接地址就可以下载文件和文件夹。

    如何通过链接分享文件或文件夹

链接方式

  • 链接
    1. 登录Huawei CloudDrive网站。
    2. 将光标悬浮在文件或文件夹上，单击右侧的

    3. 单击“访问限制”，设置提取码和有效期，单击“保存”。
      提取码是用户查看链接内容的密码。
      系统默认没有提取码，链接永久有效。
    4. 在“收件人列表”区域框中，填写收件人信息，单击“发送”。。
      果您希望通过其他方式分享链接，请单击链接右侧的复制按钮和提取码右侧的复制按钮，复制链接地址和提取码到记事本。通过其他方式将链接和提取码发送给被分享人。码。
