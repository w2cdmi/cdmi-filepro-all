如何删除StorBox中的文件

<<返回帮助中心

您也可以登录StorBox网站或云管家客户端删除文件。

通过网站删除文件

  1. 登录StorBox网站。
  2. 右键单击文件或文件夹，在右键菜单中，选择“删除”，删除文件或文件夹。
    说明

    您也可以选择文件或文件夹，在上方选择“更多>删除”，删除文件或文件夹。

    如果您想永久删除文件，请进入回收站页面删除文件。

通过云管家客户端删除文件

在云管家客户端中删除文件很简单。右键单击文件或文件夹，在右键菜单中，选择“删除”，删除文件或文件夹。

说明

您也可以选择文件或文件夹，在上方选择“更多>删除”，删除文件或文件夹。

在云管家客户端中删除的文件会出现在网站的“回收站”中。

如果您想永久删除文件，请进入回收站页面删除文件。
