StorBox Android应用

在Android手机上安装WPS Office客户端后， StorBox Android应用中可以打开以下格式的文件：

  • 文本文件（.txt）
  • Word（.doc/.docx）
  • Excel（.xls/.xlsx/.xlt/.xltx）
  • PowerPoint（.ppt/.pptx）
  • PDF（.pdf）

此外，StorBox Android应用集成了图片浏览插件，支持浏览bmp、gif、jpg、jpeg和png格式的图片。
