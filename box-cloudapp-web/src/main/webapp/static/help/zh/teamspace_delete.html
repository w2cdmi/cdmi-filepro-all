如何删除我创建的团队空间

<<返回帮助中心
  1. 登录StorBox网站。
  2. 单击左侧导航中的“团队空间”。
  3. 右键单击需要设置的团队空间，选择“解散团队”。

    您也可以单击需要删除的团队空间，在上方的菜单栏选择“解散团队”。

  4. 在“解散团队空间”的窗口中，输入“yes”。

    团队空间解散后，系统将删除团队空间中的所有数据，请谨慎操作。

  5. 单击“确定”。
