什么是StorBox同步盘客户端

<<返回帮助中心

Windows操作系统中运行的可以实现文件在本地硬盘和云端同步的客户端应用软件。您可以通过电脑中的同步文件夹访问StorBox。

除了使用StorBox网站外，建议您下载并安装StorBox同步盘客户端。

为什么呢？使用同步盘客户端方便快捷。在电脑中使用StorBox，与使用本地硬盘中的其他文件夹一样，只不过同步文件夹的内容会自动同步到与用户相关联的其他设备中。

同步盘客户端会自动监控文件夹状态，让您的文件保持同步。无论您何时何地修改了文件，只要连接网络，修改内容会自动同步到安装了StorBox的其他设备中。
