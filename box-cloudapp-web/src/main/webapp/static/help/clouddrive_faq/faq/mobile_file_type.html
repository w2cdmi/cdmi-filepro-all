我可以在手机上查看哪些类型的文件

当您查看或编辑文件时，Huawei CloudDrive Android应用会自动搜索您的设备，查找可以打开您尝试查看或编辑的文件类型的应用程序。默认情况下，Android操作系统可以打开以下文件格式：
  • 文本文件（.txt）
  • 网页（.htm/.html）
  • 图片（.jpg/.gif/.tiff）

如果Huawei CloudDrive Android应用找不到合适的应用程序打开文件，则会显示用于查看文件的Android应用程序推荐列表。

如果想进一步了解Android支持的文件格式，请查看Android网站。
