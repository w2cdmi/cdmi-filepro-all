如果我的硬盘空间无法容纳整个Huawei CloudDrive中的内容，是否可以进行选择性同步

Huawei CloudDrive默认是将所有文件同步到您的电脑中。

如果Huawei CloudDrive中存储的内容已经超出电脑硬盘的可用空间，您可以在网站上取消部分文件的同步，同时在本地删除这些文件。

  1. 登录Huawei CloudDrive网站。
  2. 将光标悬浮在文件或文件夹上，单击右侧的。

    说明：

    您也可以右键单击文件或文件夹的空白处，在右键菜单中，选择“取消同步”或者单击文件列表上方的“取消同步”。
