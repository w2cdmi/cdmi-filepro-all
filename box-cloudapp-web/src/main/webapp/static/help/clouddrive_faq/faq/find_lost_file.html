我的文件丢失了，怎样才能找回

导致文件丢失的原因多种多样，请根据下文内容逐一排查，尽快找回文件：
  • 确认丢失的文件是否是别人共享的文件

    如果文件已经消失，可能是文件拥有者将您从共享成员中移除了，也可能是删除了共享文件或者取消了共享。请联系共享拥有者确认原因。

  • 通过搜索找到文件

    请登录Huawei CloudDrive网站，在搜索框中输入丢失文件的名称。通过搜索，发现文件可能被移动到了其他位置。

  • 通过回收站找到文件

    如果文件在某个终端上被误删，由于同步功能导致其他终端上的文件也消失了，您可以登录Huawei CloudDrive网站，在回收站中查找文件。发现文件后，可以将文件恢复到原有位置。

如果通过以上方法，仍然无法找回文件，请将您所在的区域（如深圳、美国）、丢失文件的名称、创建时间和类型反馈到这里，请工程师协助找回文件。
