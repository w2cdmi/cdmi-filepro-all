如何添加或上传文件

网站

  1. 登录Huawei CloudDrive网站。
  2. 单击“上传”。

  3. 在打开的窗口中，选中需要上传的文件，单击“打开”。

    在网站右下角显示文件上传进度。上传成功后，界面右下角的上传列表框自动消失，文件已经添加到Huawei CloudDrive中。

Windows

请确保已安装Huawei CloudDrive同步客户端。电脑中会出现一个同步文件夹。该文件夹中的所有内容都会自动同步到网站和其他安装了Huawei CloudDrive的设备中。

在Windows中，将文件拖拽到同步文件夹，文件将自动被上传到云端。
