什么是Huawei CloudDrive同步客户端

Windows操作系统中运行的可以实现文件在本地硬盘和云端同步的客户端应用软件。您可以通过电脑中的同步文件夹访问Huawei CloudDrive。

除了使用Huawei CloudDrive网站外，建议您下载并安装Huawei CloudDrive同步客户端。

为什么呢？使用同步客户端方便快捷。在电脑中使用Huawei CloudDrive，与使用本地硬盘中的其他文件夹一样，只不过同步文件夹的内容会自动同步到与用户相关联的其他设备中。

同步客户端会自动监控文件夹状态，让您的文件保持同步。无论您何时何地修改了文件，只要连接网络，修改内容会自动同步到安装了Huawei CloudDrive的其他设备中。
